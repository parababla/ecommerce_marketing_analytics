# Анализ продаж интернет-магазина: исследование, проверка гипотез и прогнозирование доходов

## Описание задачи

В рамках данного проекта был проведён комплексный анализ данных интернет-магазина. Компания осуществляет продажи в четырёх регионах и использует различные рекламные каналы для привлечения клиентов.

Основная цель: определить, в какой регион и какой рекламный канал стоит инвестировать больше средств для привлечения клиентов в следующем году.

Ключевые этапы работы:
- Очистка и предобработка данных
- Исследовательский анализ (EDA)
- Проверка статистических гипотез
- Построение регрессионной модели для прогнозирования продаж
- Формулировка рекомендаций для бизнеса

## Ход выполнения работы

### Очистка и предобработка данных
- Названия столбцов приведены в соответствие с PEP8
- Удалено 13 строк с пропусками (1.3%)
- Удалено 2 полных дубликата, исправлены неявные дубликаты
- Выбросы (revenue = 1) заменены на медиану 4999
- Добавлены столбцы: "final_revenue", "day_time", "payer"

### Аналитический и графический анализ

- Построение долей продаж и пользователей по регионам, устройствам, каналам
- Анализ сезонности по месяцам, дням недели, времени суток
- Диаграмма покупок по типам оплаты

### Проведение расчётов

- Средний чек, покупки на пользователя
- Средняя продолжительность сессии по каналам и устройствам
- Топ-3 по среднему чеку: каналы, регионы, месяцы с разбивкой по регионам
- MAU по месяцам с разбивкой по каналам
- Сводная таблица по каналам (пользователи, уникальные, платящие, сумма продаж)

### Проверка гипотез

Проверено 8 гипотез с использованием критерия Краскела-Уоллиса, Z-теста и корреляции Спирмена. Все гипотезы о влиянии различных факторов (тип устройства, рекламный канал, регион, время суток) на метрики (покупки, средний чек, конверсия) не подтверждены. Взаимосвязь между длительностью сессии и суммой покупки отсутствует.

### Регрессионное моделирование

- Построена линейная регрессия для прогнозирования недельного дохода и количества покупателей
- Факторы: рекламный бюджет, комбинация "регион + канал"
- Результаты: R² = 0.84, MAE (доход) = $2 582, MAE (покупатели) = 0.5 чел.
- Выполнен прогноз годового дохода при различных стратегиях

## Результаты

- Приоритетный регион: США
- Приоритетный канал: социальные сети
